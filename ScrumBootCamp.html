プレミアムの講師報酬配分

プレミアムは4名体制で実施する

  • メイン講師(マテリアル提供、翔泳社とのやり取り、当日メイン) 70%
  • サブ講師(マテリアルを借りて一部担当) 25%
  • サポート(当日のお手伝い) 2%
  • アドバイザー (箔つけ担当、メイン講師経験者が質疑応答と懇親会参加した場合に発生) 3%

方針としては以下。上から優先

  • メイン講師には日当分は払いたい
  • サポートの人にも5000円ぐらいは払いたい
  • サブ講師は5万円ぐらいは払いたいが、勉強代も含んでていると思ってほしい
  • 集客に少し貢献できるアドバイザーもタダ働きはやめたい

Mar 22, 2013 From: nishimura@n...

現在、Scrum Boot Camp には 2 つの流れがありまーす

  • Scrum Boot Camp (地方含むコミュニティ向け休日開催のやつ)
  • Scrum Boot Camp Premium (一般企業の人向けの有料セミナーのやつ)

Scrum Boot Camp
のひとつの狙いは自分たちのリアルお仕事で研修っぽいことをするための練習台なので、メイン講師はいろんな人がバンバンやっていってほしいと思っています。

ただし、Scrum Boot Camp に求められている期待も徐々に高くなっている気がするし、終日の研修資料を作るのもシンドイのも事実です。
なので、ワークショップ大戦のようなメインで講師をやるための練習イベントをやろうと思います。
名付けて「Scrum Boot Camp Mini」です!!

今のところ、こんな感じを考えています。

  • 半日開催
  • 練習ですと明記して少人数の募集にする
  • まずは Scrum の基礎についてターゲットを絞った内容で -> 軌道に乗った後はすでに終日の研修資料を持っている人の先鋭的なネタの練習場にもしたい

また、ノーマルScrum Boot Campは品質を一定にするために云々みたいな話があったのですが、上手く機能していないので、mini
が上手く行けば、以下のような流れになるといいなーと思っています。

Scrum Boot Camp Mini(ここでスタッフからバシバシフィードバックする) -> Scrum Boot
Camp(基礎の部分のメイン講師からやってみよう) -> Scrum Boot Camp Premium

ってな感じです。
ただ、Premium は今期どうするかの話を僕が先方と話をするのを絶賛放置中なのでどうなるか分からないけど...ww
