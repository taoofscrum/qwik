「Scrum Boot Camp The Talk 増刊号」は西村さんに「大阪でもやりませんか？」ってボールを投げていた気がします?

はあくしてるよーん。
来週、翔泳社と相談してくるのでちょっと待ってね。

まだ考えがまとまってませんが、一旦ダンプしておきます。

本に関しては翔泳社もこうしていきたいという考えがあるので、道単独で考えません。
また、本の対象読者が基本的にはコミュニティ来ない人にしているので、単にコミュニティのイベントにしても仕方がないと思っています。
アジャイルサムライのようにコミュニティを生むような本ではないしww

じゃ、道で考えなくてもいいやんけ!! と思うのですが、こっちとしては道場への新規参加を増やすのに使っていきたいです。
それは、本を読んでさらに深いとこを考えたいという人へのフォロー先を知ってもらうことと、ぼくたちはその人から新しい気づきを得るメリットを増やしたいからです。

で、どうすんの? でいうとこんな感じ

  • 都内は道の集客力も高くなっているので、春あたりに常設劇場という名のファンシアターで大きく開催。大きくやる分には翔泳社も喜ぶだろうww(要確認)

-
大阪とかは可能なかぎり大手書店とコミュニティ、出版社と連携しながら開催していきたい。こっちも運営負担も少ないし、地方のコミュニティに新しい人を呼びこむきっかけになるといいなー

てな感じで思いつきでジュンク堂で生まれた Scrum Boot Camp The Talk ですが、継続的に開催していきたいという PO
の新しいワガママになりましたという話でしたww
ご意見、ご感想はいっつも募集しております。

開催概要とロゴ??つくったよーん。

6/21に「Scrum Boot Camp The Talk 春の増刊号」というイベントをやります。
このイベントは書籍「Scrum Boot Camp The
Book」を題材にScrumについて考えるイベントです。本を読んで疑問に思ったことや自分の現場でScrumをやろうとして悩むことを著者陣に直接質問したり、参加者同士で議論することによって、理解を深めていくためのイベントです。
Scrum についてよく分からないので、単に見学してみたいという方ももちろん大歓迎です。

本イベントは株式会社バンダイナムコスタジオさまのご厚意により、すばらしい会場を提供していただいております。なるべくご迷惑をかけたくないので、参加される方は時間とおりに来ていただけるようお願いします。

ぜひ、一緒に本をネタにいろんなことを語りあいませんか。
